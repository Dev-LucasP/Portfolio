Projets

Projet marathon

Ce projet web utilise Laravel pour gérer l'ensemble de l'architecture serveur. Grâce à PHP, nous avons développé des fonctionnalités, tandis que HTML et CSS structurent et stylisent l'interface utilisateur. Les MMI gèrent le front-end, tandis que nous nous occupons du back-end.

Démineur

Ce projet personnel consiste en un jeu de démineur développé avec JavaFX, en utilisant le modèle MVC pour une meilleure organisation du code. Le modèle gère la logique du jeu, comme la création de la grille et la gestion des mines. Le contrôleur coordonne les interactions entre l'interface et la logique, tandis que la vue s'occupe de l'affichage des informations à l'utilisateur.

Pokédex

Ce projet de Pokédex est développé en Python et utilise des requêtes SQL pour gérer une base de données contenant des informations sur les Pokémon. Les données sont récupérées et manipulées via des requêtes SQL, permettant de stocker, rechercher, et filtrer les Pokémon selon différents critères (type, bruit, statistiques, etc.). L'application offre une interface utilisateur simple qui permet d'afficher les résultats des requêtes en temps réel.
